# Guia do Repositório

Este guia apresenta a organização do repositório de microcontroladores. Use-o para navegar pela estrutura de pastas e localizar rapidamente materiais de estudo, templates, projetos e ativos compartilhados.

- `docs/`: Documentos de referência, mapas de estudo e glossário.
- `templates/`: Modelos para documentar projetos, chips e anotações de aula.
- `aulas/`: Anotações consolidadas das aulas mais recentes.
- `chips/`: Documentação específica, datasheets e exemplos de código para cada família.
- `projetos/`: Projetos em desenvolvimento com firmware, esquemas e testes.
- `assets/`: Recursos visuais centralizados (imagens, esquemas e PCBs).

Atualize este arquivo sempre que a estrutura mudar de forma significativa.
